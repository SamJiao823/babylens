# 📝 Indie Hackers — 1条（长篇）

## 第16条：完整故事

> 打开 indiehackers.com → 注册 → Write a story → 粘贴

```
Title: I built an AI baby face generator in 8 days — here's everything I learned

Body:

I've been reading Indie Hackers for months, and I finally built something worth sharing.

The idea came to me on a Thursday night: "What if couples could upload their photos and see what their future baby would look like?"

By Friday morning I was coding.

Tech Stack:
• Frontend: Vanilla HTML/CSS/JS (no framework, wanted it fast)
• Hosting: Cloudflare Pages (free tier, global CDN)
• AI: Stability AI SDXL (img2img)
• Payments: PayPal

The hardest technical challenge was getting the AI to actually change the clothes. At first, it kept generating babies wearing adult clothes — which is creepy. I had to dial image_strength down to 0.02 and add strong negative prompts to fix it.

Another challenge: SDXL has a minimum steps requirement of 10. I initially set it to 5 thinking it would be faster, but the API just returned errors. Lesson learned: read the docs first.

The product is live at babylens.pages.dev — $9.99 one-time payment, no subscription.

What I learned:
1. Shipping fast beats perfect. I could've spent weeks on React/Vue. Instead I wrote raw JS and shipped in 8 days.
2. API costs matter. Each image costs ~$0.02. At $9.99, that leaves decent margin.
3. The viral loop is built-in: people WANT to share their baby photos. They send them to family, post on social media. Zero ad spend so far.

Would love feedback from this community. What would you do differently?
```

Title: How I Built an AI Baby Face Generator in 8 Days with Vanilla JS and Cloudflare Pages

Body:

I wanted to see if I could build and ship a working AI product in a week. Here's how it went.

The Idea
Upload photos of both parents → AI generates a realistic baby face prediction. Simple concept, but the execution had some surprises.

Tech Stack
- Frontend: Vanilla HTML/CSS/JS
- Hosting: Cloudflare Pages (free tier)
- AI: Stability AI SDXL API
- Payments: PayPal

The Hard Parts
1. Getting the AI to generate baby clothes instead of adult clothes. Solution: image_strength: 0.02 + negative prompts like "adult, grown-up clothes, suit, dress"
2. SDXL requires minimum steps of 10. Setting it lower just returns errors.
3. Cloudflare Workers have a 30-second timeout. The API call takes 8-10 seconds, so we're fine, but worth noting.

The Result
Live at babylens.pages.dev — $9.99 one-time payment.

Key Takeaway
You don't need a complex framework or months of development to ship an AI product. Vanilla JS + a good API + Cloudflare's free tier is enough to get started.
